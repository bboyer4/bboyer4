Pennsylvania Presidential Election Map 2020

This is a map displaying the results of the 2020 United States Presidential Election in Pennsylvania. The map was created using ArcGIS Pro and shows whether each county in Pennsylvania voted for Joe Biden and Donald Trump and what the percentage was. Counties that voted for Trump are in red and counties that voted for Biden are in Blue with the saturation of each color indicating the percentage of the vote. This map aims to provide a clear visual of the voting percentage breakdown in Pennsylvania during the 2020 election by county.

To create this map, I joined data containing the geography of PA Counties with a County Voting Data Table. I then made a choropleth map using bivariate colors. The light red-dark red color scheme indicates Trump voting percentages and the light blue-dark blue color scheme indicates the percentage of votes for Biden with darker saturation indicating a higher percentage and lighter saturation indicating lower.

Chicago Police District Homicide Density Map

This map is a choropleth map displaying the homicide density in Chicago by police district. The problem this map aimed to address was to see how homicides could impact tourism in Chicago. The first objectives was to describe the spatial distribution of the density of homicides by police district in Chicago The second objective was to compare the spatial patterns of homicides, Airbnb listings, and Airbnb prices in Chicago

To create this map, I first had to create a field containing the value of Area by square mile. Once I calculated the area, I joined the table containing this value to the tables containing AirBnb review and homicide data. I then used the Calculate Field option to calculate the density of Airbnb reviews and homicides by police district.
